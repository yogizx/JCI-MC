# JCI Madurai - Member Portal

A premium React-based web application for JCI Madurai, featuring a modern dashboard, member profiles, and membership management.

## 🚀 Features

- **Modern UI/UX**: Built with React and Tailwind CSS for a premium, responsive experience.
- **Member Portal**: comprehensive profile management for members, including business and family details.
- **Admin Dashboard**: Specialized routing for administrative tasks.
- **Production Ready**: Optimized for deployment on platforms like Vercel, Netlify, or shared hosting (Apache).

## 🛠️ Tech Stack

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yogizx/JCI.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## 🚀 Deployment

### Vercel / Netlify
This project is pre-configured for Vercel with `vercel.json`. Simply connect your GitHub repository and it will deploy automatically.

### Shared Hosting (BigRock/Bluehost)
1. Run `npm run build`.
2. Upload the contents of the `dist` folder to your `public_html` directory.
3. Ensure the `.htaccess` file is present in the root to handle client-side routing.
